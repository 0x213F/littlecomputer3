Little Computer 3

Learn assembly with a free IDE build on the web

LC3 (Little Computer 3) is an educational assembly programming language. If this is your first time coding or you have experience with a high-level programming language, there are many benifits to learning how assembly code is written.

Efficient

It runs directly on the proccessor.

Low Level

Visualize the memory stack.

Code

Program in either binary or assembly.

Keyboard

Input for programs.

Foundation

Great for aspiring computer engineers.

I/O

Terminal access to print results.

The advanced debugger allows you to view the memory dump in any format.

Peek into the contents of each of the eight general purpose registers, program counter, and instruction regsiter.

Change the color scheme of the editor.
